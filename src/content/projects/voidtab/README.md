## 项目简介

VoidTab 是一个基于 Vue 3 + TypeScript 构建的极简、赛博朋克风浏览器起始页（New Tab Extension）。

它摒弃传统导航页的臃肿与广告，强调极致视觉与高效操作，提供 GUI/CLI 双模、沉浸式 HUD、毛玻璃 UI 与多端数据同步能力。

## 当前进展

- 已上架 Edge 插件
- 已有 10 位体验用户
- 持续更新迭代中

安装方式：在 Edge 插件商店搜索 `Edgevoidtab` 即可安装使用。

## 关键能力

- 毛玻璃拟态 UI + 动态主题
- GUI/CLI 双模切换与终端化交互
- 多种效率/娱乐/开发者组件
- AI 智能助手面板
- WebDAV 同步、书签导入、拖拽管理、聚合搜索
