## 项目设定

核心玩法为：

- 魔塔式成长框架
- 日程牌构筑（Schedule Deckbuilding）
- 武侠 Roguelite 冒险
- 宗门经营沙盒

每天早晨抽到的一手牌，决定当天可执行的行动：练功、种田、建造、招人、出行、交涉。

## 长线目标

在季节事件与江湖势力博弈中经营宗门，最终进入宗门大战。
