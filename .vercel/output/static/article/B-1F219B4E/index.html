这是一篇测试 MDX 功能与进阶语法的文章，你可以在其中混合使用标准 Markdown 语法和我们刚刚封装的 React/Astro UI 组件。由于我们修改了前置配置（Frontmatter），现在不需要再手动填写 pubDate、draft、encrypted 等无聊的默认属性了！
